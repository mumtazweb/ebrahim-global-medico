Global Medico Admissions - Static HTML Hostinger Upload Notes

Live domain:
https://globalmedico.in

Local preview:
Open this folder in VS Code and use the Live Server / Go Live extension.
Start from index.html.

Upload instructions:
1. Open Hostinger hPanel File Manager.
2. Go to public_html.
3. Upload the website files and folders inside this project, not the parent folder itself.
4. If using the ZIP package, upload global-medico-html-upload.zip into public_html and extract it there.
5. After extraction, index.html should be directly inside public_html.

Edit locations:
- Homepage content: index.html
- About page: about.html
- Countries page: countries.html
- University page: universities.html
- Contact page: contact.html
- Apply page: apply.html
- Colors/layout/mobile styling: assets/css/style.css
- JavaScript interactions: assets/js/main.js
- Logo: assets/images/globalmedigologo.png
- Hero doctor/student image: assets/images/hero-doctor.png
- Country images: assets/images/country-*.jpg

Forms:
This static HTML version does not save enquiries on the server.
The contact and apply forms open WhatsApp with the entered enquiry details.

No server-side code, database, or build step is required for hosting.
